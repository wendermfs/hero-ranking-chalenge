# 🏆 Hero Ranking Challenge

## 🇧🇷 Português

Projeto desenvolvido como desafio de lógica de programação.

## 📌 Objetivo

Criar um sistema que classifique o nível de um herói com base na quantidade de experiência (XP).

## 🚀 Tecnologias utilizadas

* JavaScript
* Node.js
* VS Code
* Git e GitHub

---

# 📖 Regras de classificação

| XP do Herói             | Nível      |
| ----------------------- | ---------- |
| Menor que 1.000         | Ferro      |
| 1.001 até 2.000         | Bronze     |
| 2.001 até 5.000         | Prata      |
| 5.001 até 7.000         | Ouro       |
| 7.001 até 8.000         | Platina    |
| 8.001 até 9.000         | Ascendente |
| 9.001 até 10.000        | Imortal    |
| Maior ou igual a 10.001 | Radiante   |

---

# 💻 Exemplo de saída

```bash
O Herói de nome Arthur está no nível Ouro
```
